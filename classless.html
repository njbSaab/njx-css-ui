FAQ

Frequently asked questions

Everything you need to know about njX UI Classless Mode.

What is Classless Mode?

Classless Mode styles semantic HTML elements (h1–h6, p, table, form, etc.) directly — no class names needed. It's ideal for content-heavy pages, Markdown output, CMS templates, or anywhere you want beautiful defaults without touching your markup.

Can I use Classless Mode and Full UI Mode together?

No — use one mode per page. Both files define styles for the same elements and combining them may cause conflicts. Classless Mode (classless.min.css) is intentionally separate from Full UI Mode (style.min.css).

How do I switch themes?

Set data-theme on the <html> element:

<html data-theme="light">   <!-- or dark, red, blue, green… -->

Or switch at runtime with JavaScript:

document.documentElement.setAttribute('data-theme', 'purple');

If no data-theme is set, Classless Mode automatically follows prefers-color-scheme.

Is Classless Mode accessible?

Yes. Focus styles use a visible outline on interactive elements. Color contrast ratios are designed to pass WCAG AA for both light and dark themes. Form fields use accent-color for native checkboxes and radio buttons.

How small is the CSS file?

The minified bundle (classless.min.css) is approximately 20 KB — lighter than most images. It includes all design tokens, typography, forms, tables, code blocks, and navigation styles.

Can I customize the tokens?

Yes. Override any CSS variable in your own stylesheet after including classless.min.css:

:root {
  --color-primary: #ff6b35;
  --font-sans: "Georgia", serif;
  --radius-md: 4px;
}
