🗂 Sidebar
Drawer/sidebar panel — 4 variants: left overlay, right overlay, left push, fullpage. Controlled via sidebarOpen(id) / sidebarClose(id) / sidebarToggle(id).
Left — overlay (default)
<!-- Trigger -->
<button class="btn btn-primary" onclick="sidebarOpen('my-sidebar')">Open</button>

<!-- Backdrop -->
<div class="sidebar-backdrop" id="my-sidebar-backdrop" onclick="sidebarClose('my-sidebar')"></div>

<aside class="sidebar" id="my-sidebar">
  <div class="sidebar-header">...</div>
  <div class="sidebar-body">...</div>
</aside>
Right — overlay
<aside class="sidebar sidebar-right" id="my-sidebar-right">
  ...
</aside>
Left — push content
<!-- Wrap page in .sidebar-layout -->
<div class="sidebar-layout">
  <aside class="sidebar sidebar-push" id="push-sidebar">...</aside>
  <div class="sidebar-content">...</div>
</div>
Fullpage — full viewport
<aside class="sidebar sidebar-fullpage" id="fullpage-menu">
  <div class="sidebar-header">...</div>
  <div class="sidebar-body">...</div>
</aside>
JS API
sidebarOpen('id') sidebarClose('id') sidebarToggle('id')
sidebarOpen('my-sidebar')    // open
sidebarClose('my-sidebar')   // close
sidebarToggle('my-sidebar')  // toggle open/close
